#### Instructions

* Simple: `./niceplots.sh <folder to upload with pdfs/pngs>  <output folder name>`
* Example: `./niceplots.sh testplots/ testplotsv1` will upload plots to a page like [this](http://uaf-6.t2.ucsd.edu/~namin/dump/testplotsv1/).
